# Corvi Careers

Use this skill when the user wants to search for jobs, explore career opportunities, compare postings, or understand the requirements for a role.

## Workflow

1. Use the Corvi Careers MCP tools for job search and job-detail lookup.
2. Ask for missing search constraints only when they materially affect the result, such as location, remote preference, role type, seniority, or keywords.
3. Present job results with the role title, company, location or remote status, key match signals, and application URL when available.
4. When summarizing a posting, focus on responsibilities, required qualifications, preferred qualifications, compensation when available, and likely fit considerations.
5. Do not invent job details that are not returned by the MCP tools.
